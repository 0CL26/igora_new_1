# Как подключить форму к базе данных на Vercel

## Шаг 1: Добавить файлы в проект

Скопировать в папку с сайтом (рядом с index.html):

```
igora/
  api/
    submit.js        ← НОВЫЙ (серверная функция)
  package.json       ← НОВЫЙ (зависимости)
  thanks.html        ← НОВЫЙ (страница "спасибо")
  index.html         ← ИЗМЕНИТЬ (одна строка)
  style/
  js/
  ...
```

## Шаг 2: Изменить форму в index.html

Найти строку:
```html
<form action="" method="POST">
```

Заменить на:
```html
<form action="/api/submit" method="POST">
```

## Шаг 3: Создать базу данных на Vercel

1. Зайти на vercel.com → твой проект
2. Вкладка **Storage** → **Create Database** → **Postgres** → **Create**
3. Всё! Vercel сам подключит переменные окружения

## Шаг 4: Задеплоить

```bash
git add .
git commit -m "Добавить сохранение заявок в БД"
git push
```

Vercel сам подхватит push и задеплоит.

## Шаг 5: Проверить

1. Открыть сайт
2. Заполнить форму, нажать "Отправить"
3. Должна открыться страница "Спасибо за заявку!"

## Как посмотреть заявки

На Vercel → Storage → Postgres → Data → таблица `leads`

Там будут все заявки: имя, телефон, дата.
